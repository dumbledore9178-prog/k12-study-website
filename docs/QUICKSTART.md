# Quick Start Guide - K12 Study Website\n\n## вҡЎ 5-Minute Setup\n\n### 1. Prerequisites\n\n```bash\n# Check Node.js version (need 18+)\nnode --version\n\n# Check npm\nnpm --version\n```\n\n### 2. Clone & Install\n\n```bash\n# Clone repository\ngit clone https://github.com/dumbledore9178-prog/k12-study-website.git\ncd k12-study-website\n\n# Install dependencies\nnpm install\n```\n\n### 3. Setup Environment\n\n```bash\n# Copy example environment file\ncp .env.example .env.local\n\n# Edit .env.local with your values (see below)\n```\n\n### 4. Get Google OAuth Credentials\n\n1. Go to [Google Cloud Console](https://console.cloud.google.com/)\n2. Create a new project\n3. Go to Credentials вҶ’ Create OAuth 2.0 credentials\n4. Add `http://localhost:3000/api/auth/callback/google` to authorized redirects\n5. Copy Client ID and Client Secret\n6. Paste into `.env.local`\n\n### 5. Setup Database\n\n**Option A: PostgreSQL Local**\n```bash\n# Create database\ncreatdb k12_study\n\n# Update DATABASE_URL in .env.local\nDATABASE_URL=\"postgresql://postgres:password@localhost:5432/k12_study\"\n```\n\n**Option B: PostgreSQL Docker**\n```bash\n# Run PostgreSQL container\ndocker run -d \\\n  --name k12-postgres \\\n  -e POSTGRES_PASSWORD=password \\\n  -e POSTGRES_DB=k12_study \\\n  -p 5432:5432 \\\n  postgres:15\n\n# DATABASE_URL in .env.local\nDATABASE_URL=\"postgresql://postgres:password@localhost:5432/k12_study\"\n```\n\n### 6. Initialize Database\n\n```bash\n# Run migrations\nnpx prisma migrate dev\n\n# Seed sample data (optional)\nnpx prisma db seed\n```\n\n### 7. Start Development Server\n\n```bash\nnpm run dev\n```\n\n**Access the app**: http://localhost:3000\n\n---\n\n## рҹ“Ӣ Environment Variables Template\n\n```env\n# Database (PostgreSQL)\nDATABASE_URL=\"postgresql://user:password@localhost:5432/k12_study\"\n\n# NextAuth\nNEXTAUTH_URL=\"http://localhost:3000\"\nNEXTAUTH_SECRET=\"generate-with: openssl rand -base64 32\"\n\n# Google OAuth (from Google Cloud Console)\nGOOGLE_CLIENT_ID=\"your-client-id-here\"\nGOOGLE_CLIENT_SECRET=\"your-client-secret-here\"\n\n# App Configuration\nNEXT_PUBLIC_APP_ENV=\"development\"\nNEXT_PUBLIC_APP_NAME=\"K12 Study\"\n```\n\n**Generate NEXTAUTH_SECRET:**\n```bash\nopenssl rand -base64 32\n```\n\n---\n\n## рҹ§Ә Test the App\n\n### 1.
